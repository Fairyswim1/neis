🧚 나이스 복붙 요정

엑셀 → 나이스, 요정이 대신 복붙!

① 메뉴 선택 → ② 엑셀 파일 또는 복사 붙여넣기 → ③ 전체 입력 → ④ 나이스 입력칸 클릭 (스토어 설치 직후는 F5)

(1 = 첫 번째 행)
비우면 마지막 열까지

입력 설정

나이스 화면마다 Tab·Enter 횟수가 다릅니다. 보통은 위에서 메뉴를 고르면 됩니다. 이 설정은 화면이 프리셋과 다를 때만 바꾸세요.

동작 순서 (한 학생)

  1. 첫 칸에 값 입력
  2. 같은 학생 다음 칸이 있으면 → 셀 간 Tab 만큼 이동
  3. 그 학생 입력이 끝나면 → 다음 행 이동 방식으로 다음 학생으로

항목 설명

셀 간 Tab 횟수
한 학생 안에서 옆 칸으로 넘어갈 때 Tab을 몇 번 누를지. 칸이 1개면 1, 진로처럼 4칸이면 보통 1(칸마다 1번).
행 끝 Tab 횟수
한 학생 입력 후 다음 학생 첫 칸까지 Tab을 몇 번 누를지. 「Tab — 행 끝 Tab 횟수만큼」을 선택했을 때만 사용.
다음 행 이동
  • Enter — Enter만 눌러 다음 줄·다음 학생 (행 끝 Tab 무시)
  • Tab — Tab만 행 끝 Tab 횟수만큼 (세특·행동특성·지필 등 대부분)
  • Enter 후 Tab — Enter 먼저, 그다음 Tab 추가 (특수 화면만)
입력 간격 (ms)
칸마다 기다리는 시간. 기본 70. 긴 글은 자동으로 더 기다립니다. 건너뛰면 100~150으로 올려 보세요.

Tab 횟수 찾는 법

  1. 나이스에서 1번 학생 입력칸 클릭
  2. 키보드 Tab을 직접 누르며 2번 학생 첫 칸까지 세기
  3. 그 횟수를 행 끝 Tab 횟수에 입력 · 「Tab — 행 끝 Tab 횟수만큼」 선택
  4. 한 행 입력으로 1명만 테스트 후 전체 입력

메뉴별 참고 (프리셋 값)

  • 세특·독서·행동특성 — 행 끝 Tab 2
  • 진로희망 — 행 끝 Tab 1 · 엑셀 4열/학생
  • 자율활동 — 행 끝 Tab 3
  • 지필평가 — 행 끝 Tab 6
  • 수행평가 — Enter (Tab 0)
