# 나이스 복붙 요정 개인정보 처리방침

**최종 업데이트:** 2026년 6월 16일  
**제작:** IASA HJ  
**문의:** GitHub 저장소 Issues (https://github.com/Fairyswim1/neis/issues)

---

## 1. 개요

「나이스 복붙 요정」(이하 "본 확장 프로그램")은 교사가 엑셀 파일의 데이터를 나이스(NEIS) 교육행정정보시스템 입력 화면에 자동으로 복붙하는 Chrome 확장 프로그램입니다.

본 개인정보 처리방침은 본 확장 프로그램이 사용자 정보를 어떻게 처리하는지 설명합니다.

---

## 2. 수집하는 정보

본 확장 프로그램은 **사용자의 개인정보를 수집·전송·판매하지 않습니다.**

다음 데이터는 **사용자 기기(브라우저) 내부에서만** 처리됩니다.

| 데이터 | 용도 | 저장 위치 | 외부 전송 |
|--------|------|-----------|-----------|
| 엑셀 파일 내용 | 나이스 입력 화면에 복붙 | 브라우저 메모리 (일시적) | 없음 |
| 메뉴·Tab 설정 | 다음 사용 시 설정 유지 | Chrome 로컬 저장소 | 없음 |
| 입력 진행 상태 | 화면에 상태 표시 | Chrome 로컬 저장소 | 없음 |

본 확장 프로그램은 별도의 회원가입, 로그인, 서버 업로드를 요구하지 않습니다.

---

## 3. 정보 처리 방법

1. 사용자가 직접 선택한 엑셀(`.xlsx`, `.xls`, `.csv`) 파일을 **브라우저에서만** 읽습니다.
2. 읽은 데이터는 사용자가 지정한 나이스(NEIS) 입력칸에 **자동 입력**됩니다.
3. 엑셀 데이터는 **개발자 서버 또는 제3자 서버로 전송되지 않습니다.**
4. 입력 설정(메뉴 프리셋, Tab 횟수 등)은 Chrome `storage` API를 통해 **사용자 브라우저에만** 저장됩니다.

---

## 4. 요청 권한 및 사용 목적

| 권한 | 사용 목적 |
|------|-----------|
| `storage` | 사용자 설정(메뉴, Tab 설정) 저장 |
| `activeTab` | 사용자가 실행한 나이스 탭에서만 동작 |
| `scripting` | 나이스 입력칸에 데이터 입력 |
| `webNavigation` | iframe 내부 입력칸 위치 확인 |
| `sidePanel` | 사이드 패널 UI 표시 |
| `*.neis.go.kr`, `*.eduptl.kr` 등 | 나이스 시스템 페이지에서만 동작 |

본 확장 프로그램은 나이스 관련 페이지 외의 사이트에서 사용자 데이터를 수집하거나 변경하지 않습니다.

---

## 5. 제3자 제공

본 확장 프로그램은 사용자 정보를 **제3자에게 제공·공유·판매하지 않습니다.**

본 확장 프로그램은 Google Chrome Web Store를 통해 배포되며, Chrome Web Store의 정책이 별도로 적용될 수 있습니다.

---

## 6. 데이터 보관 및 삭제

- 엑셀 데이터: 입력 작업이 끝나면 브라우저 메모리에서 사라집니다.
- 설정 데이터: Chrome 확장 프로그램 **제거** 시 함께 삭제됩니다.
- 사용자는 Chrome 설정 → 확장 프로그램 → 나이스 복붙 요정 **제거**로 모든 로컬 데이터를 삭제할 수 있습니다.

---

## 7. 아동·학생 정보

본 확장 프로그램은 교사가 **직접 준비한 엑셀 파일**을 나이스에 입력하는 도구입니다.  
학생 개인정보의 처리 책임은 **엑셀 파일을 준비·사용하는 교사 및 소속 학교**에 있습니다.

본 확장 프로그램 제작자는 사용자가 입력하는 데이터의 내용에 접근·저장하지 않습니다.

---

## 8. 보안

- 모든 데이터 처리는 사용자 **로컬 브라우저** 내에서 이루어집니다.
- 외부 API 호출, 원격 코드 실행, 분석·추적 도구를 **사용하지 않습니다.**
